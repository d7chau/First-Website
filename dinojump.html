DINO-JUMP

A single-player game created in the OpenGL framework LibGDX

Dino-Jump is a single player game developed in the framework LibGDX as a collaborative two-man effort for our computer science class during highschool. It was built within a two month timespan as a final project based on our knowledge learned previously through the semester. Unfortunately, features of the game such as platforms constantly randomly generating were envisioned but not able to implement smoothly.

The user is able to move the dinosaur sprite left and right by using the left and right directional keys. The goal of the game is to reach the highest score possible by jumping on platforms and not falling through the bottom of the screen (jumping is automatic and not activated by any keys). If this were to occur, the game would then be over.
