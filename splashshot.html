Splash Shot

A single-player Java game created using IntelliJ IDEA

Splash Shot is a single player Java game developed using the IntelliJ IDEA as a collaborative two-man effort for a computer science class during highschool. It was built within a semester as a final project based on our knowledge learned previously.

The objective of Splash Shot is to successfully land a basketball into a basketball net. By dragging the basketball back, a bar will fill up telling you how much power will be placed into the shot. After successfully landing the basketball into the basket, your score will increment by one and another basket will proceed to spawn above the previous one at a different angle to increase difficulty. This would continue so forth. If you were to miss, the current game would be over and your score would reset back to zero. The objective of the game is to obtain the highest score possible.
