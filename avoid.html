AVOID

A two-player game created in the language Processing

Avoid is a two player game developed in the language Processing (simplified version of Java) as a collaborative two-man effort for a computer science class during highschool. It was built within a two month timespan as a final project based on our knowledge learned previously through the semester. Features such as having the sprites rotate 90 degrees on key command were unfortunately not able to be implemented in the game as envisioned.

One player is able to use WASD, while the other is able to use the directional keys to move their sprite in the corresponding direction. The goal of the game is to land the side of your square sprite that contains a black rectangle to any of the sides that the opponent has that does not contain a black rectangle OR bounce the other player's square sprite off of the screen by colliding two sides containing black rectangles / two blank sides..
